Web Scrobbler
Scrobble music all around the web!

Web Scrobbler helps music listeners to scrobble their online playback history.

Key features

  • Support for Last.fm, Libre.fm, and ListenBrainz
  • Now playing notifications
  • Built-in track metadata filter
  • A lot of websites supported

Browsers

The extension can be installed on Chrome, Chrome-based browsers (Opera, Vivaldi etc.), and Firefox.

Websites

We currently support scrobbling to different websites. With more being added with each release.
